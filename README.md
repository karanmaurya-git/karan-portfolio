# Karan Kumar Maurya — Portfolio

## Run in VS Code (2 ways)

---

### ✅ Method 1 — Live Server Extension (Easiest)

1. Open VS Code
2. Open this folder: `File → Open Folder → select karan-portfolio`
3. Install extension: search **"Live Server"** by Ritwick Dey → Install
4. Right-click `index.html` → **Open with Live Server**
5. Opens at → http://127.0.0.1:5500

---

### ✅ Method 2 — Terminal (npm)

Open VS Code terminal (`Ctrl + backtick`) and run:

```bash
npm install
npm start
```

Opens at → http://localhost:3000

---

### ✅ Method 3 — No install needed

Just double-click `index.html` — opens directly in your browser.
